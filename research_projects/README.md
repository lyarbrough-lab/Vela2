## Project Philosophy

Each project follows a consistent structure:

```
project_name/
├── manuscript/          # LaTeX or markdown manuscript
├── figures/            # Diagrams, plots, visualizations
├── simulations/        # Code and simulation results
├── data/              # Experimental or processed data
├── references/        # Bibliographic materials
├── paper.pdf          # Generated PDF (when available)
└── README.md          # Project documentation
```
