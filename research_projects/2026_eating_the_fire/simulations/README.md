# Simulation Files

This directory contains Python scripts and data files for the gamma harvesting simulations described in Paper I ("Eating the Fire") and related research.

## Core Gamma Harvesting Simulations

### `thin_foil_simulator.py`
**Purpose:** Monte Carlo simulation of gamma interaction in graded-Z multilayer stacks.

**Key features:**
- Tracks gamma photons through alternating high-Z (Pb, W) and medium-Z (Cu) layers
- Models Compton scattering, photoelectric absorption, and pair production
- Calculates charge collection efficiency based on electron escape depth
- Outputs efficiency vs layer thickness, number of layers

**Usage:**
```bash
python thin_foil_simulator.py
```

### `simple_simulator.py`
**Purpose:** Simplified version for debugging and concept validation.

**Key features:**
- Basic photon tracking with absorption probabilities
- Quick efficiency calculations
- Used for initial geometry optimization

### `debug_simulator.py`
**Purpose:** Debugging version with extensive logging.

**Key features:**
- Step-by-step photon tracking
- Detailed interaction logging
- Used to identify and fix efficiency calculation bugs

### `thickness_sweep.py`
**Purpose:** Systematic thickness parameter study.

**Key features:**
- Sweeps layer thickness from 1 μm to 1 cm
- Calculates efficiency for each thickness
- Identifies optimal thickness range (100 μm sweet spot)
- Generates data for `thickness_results.txt`

## Dark Matter Analysis (Related Research)

### `dark_matter_analysis.py`
**Purpose:** Analysis of dark matter halo distributions from CAMELS simulation.

**Key features:**
- Loads halo catalogs from `https://users.flatironinstitute.org/~camels/`
- Calculates mass function and power law exponent
- Generates comparison plots with emotion vector data

**Usage:**
```bash
python dark_matter_analysis.py
```

### `download_dark_matter_data.py`
**Purpose:** Downloads and processes dark matter halo data.

**Key features:**
- Fetches halo catalogs from Flatiron Institute server
- Extracts halo masses, positions, velocities
- Saves processed data for analysis

### `geometry_results.json`
**Output:** Comparison metrics between dark matter halos and emotion semantics.

## Simulation Methodology

### Physics Models
1. **Gamma interactions:**
   - Compton scattering: Klein-Nishina cross-section
   - Photoelectric absorption: NIST XCOM data interpolation
   - Pair production: Bethe-Heitler cross-section

2. **Electron transport:**
   - Continuous slowing down approximation (CSDA)
   - Escape depth: 2-10 μm for charge collection
   - Energy deposition profile

3. **Geometry:**
   - Multilayer stack with alternating materials
   - Interface counting for surface-near interactions
   - Boundary conditions for photon escape

### Key Parameters
- **Gamma energy:** 2 MeV (representative spent fuel spectrum)
- **Materials:** Pb (high-Z), Cu (medium-Z), W (alternative high-Z)
- **Layer thickness:** 1 μm to 1 cm range studied
- **Stack height:** 2.6 cm (standard dimension)
- **Electron escape depth:** 10 μm baseline, 2-10 μm sensitivity study

## Results Summary

### Efficiency Findings
1. **Bulk absorption:** 0.069% efficiency (1.3 cm Pb+Cu)
2. **100 μm layers:** 1.831% efficiency (26.5× improvement)
3. **5 μm layers:** 8.742% efficiency (126× improvement, impractical)
4. **Optimal practical:** 100 μm layers, 260 layers, 1.8% efficiency

### Geometry Advantage
The improvement comes from increasing interface count:
- **Bulk:** 1 interface (front surface)
- **100 μm layers:** 260 interfaces
- **Each interface** allows photoelectron generation within escape depth

### Pressure Test Results
- **Electron range sensitivity:** Efficiency 0.9-3.2% depending on assumptions
- **Geometry advantage robustness:** 8-25× improvement across parameter range
- **Manufacturing tolerance:** ±20% thickness variation reduces efficiency <10%

## Usage Notes

### Dependencies
```bash
pip install numpy matplotlib scipy
```

### Running Simulations
```bash
# Basic efficiency calculation
python thin_foil_simulator.py

# Thickness optimization study  
python thickness_sweep.py > results.txt

# Dark matter analysis
python download_dark_matter_data.py
python dark_matter_analysis.py
```

### Output Files
- `.txt` files: Numerical results
- `.json` files: Structured data
- Plots generated by scripts (saved as `.png`)
